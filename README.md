# JustP2P

**JustP2P is a decentralized chat platform that uses peer-to-peer connections instead of centralized servers—privacy-first and secure.**

## 🚀 Features

- 🔒 End-to-end peer-to-peer messaging
- 🛡️ No central servers or data collection
- 🌐 Works directly in the browser
- 🧩 Open-source and GPLv3 licensed

## 🛠️ Tech Stack

- **Frontend**: React
- **Backend**: Node.js + Express
- **Database**: MongoDB (optional for metadata)
- **P2P Layer**: WebRTC

## 📦 Installation

bash
git clone https://github.com/mustafa4411/justp2p.git
cd justp2p
npm install
npm start

## 🧪 Development
npm run dev
